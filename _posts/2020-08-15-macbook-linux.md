title: "Linux on the MacBook Pro (mid-2009)"
date: 2020-08-15
layout: post
---

A few months ago, I bought a mid 2009 MacBook Pro for only £125 with free postage on eBay, and I have to say, for a mid 2009 laptop, 8GBs RAM and a Kingston SSD, it makes for a great development machine. Sadly, the latest macOS version available (without using unofficial patches) is El Capitan 10.11. This gave me the incentive to make the machine a dual boot development machine capable of using macOS software (32-bit and 64-bit software - beat that Catalina!) such as Ableton Live 10, Final Cut Pro and also some more development-heavy applications such as an updated GCC compiler, VirtualBox/QEMU for homebrew/mainstream OS kernel development with the latest development libraries and headers available. Coming from a background of using mainly Wintel computers and keyboards, the MacBook having the Eject key instead of Delete, and Super/Windows key and Alt/Option keys swapped is really starting to get on my nerves now. Everything else about it is great however. Thankfully the open nature of Linux (Xubuntu in my case, which has always been my go-to distribution for new hardware) gives me the ability to remap the keys to behave much more like a bog-standard PC keyboard, which I feel will aid in my productivity and development, even if it's a miniscule change it will just feel a lot better.
